# Test Event

Test how an event would be processed by the rule engine.

**Endpoint:** `POST /api/v1/events/test`

**Auth:** JWT + Admin

**Query Parameters:**
- `dry_run` (optional) - true/false, default: true

See [Workflow: Test Event](../workflows/test-event-dryrun/) and [Workflow: Execute Event](../workflows/test-event-execute/) for detailed documentation.
